# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Purpose

An awesome-list style repository tracking the rapidly evolving AI engineering landscape: tools, libraries, practices, articles, videos, and resources.

## Format

All content lives in README.md using the awesome-list convention:
- Organized by category with clear headings
- Entry format: `- [Name](link) - Description (optional) \`YYYY-MM-DD\``
- Always use the current date (today's date) when adding entries
- Link to primary sources (official docs, GitHub repos, original articles)
