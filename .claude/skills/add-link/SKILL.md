# Add Link Skill

Add a new link to the AI Dev awesome-list.

## Usage

```
/add-link <url>
```

## Instructions

1. Fetch the URL to understand what the resource is about
2. Read README.md to see current categories and entries
3. Determine the best category for the link:
   - **Claude Code Plugins & Skills** - Plugins, skills, or extensions for Claude Code
   - **MCP Servers** - Model Context Protocol servers
   - **Tools** - Developer tools, applications, utilities
   - **Libraries & Frameworks** - Code libraries and frameworks
   - **Articles** - Blog posts, tutorials, written content
   - **Alpha Leak** - High-value tweets, threads, insider tips
   - **Videos** - Video content
   - **Courses** - Educational courses
   - **Research Papers** - Academic papers

4. Add the entry in the format: `- [Name](link) - Description \`YYYY-MM-DD\``
   - Use today's date
   - Keep descriptions concise (under 100 characters)
   - Link to the primary source

5. If the user says "commit" or "push", commit and push the changes
